三步，比搜快十倍

1. 写中文

想干嘛写啥：`docker 清理无用镜像和停止容器`。不用学，本来就是母语。

2. 按快捷键
Ctrl+G 极速 · 直接出一条命令
Ctrl+/ 选择 · 三个候选各带解释
3. 看懂了回车

命令不是黑盒子。先看清楚它干啥，再决定跑不跑。不爽还能换一个。

一分钟装好

`autosh` 装完就能用，短命令叫 `ah`。配好 API Key，煞笔终端立马听懂中文。 

# 一键安装
curl -fsSL https://wang-h.github.io/autosh/install.sh | bash

# 或手动装
pip install autosh       # 或者: uv tool install autosh

# 配置 API Key + 切模型
autosh config set api_key <your-api-key>
autosh config set provider deepseek   # kimi / minimax / qwen / glm

# 安装 Shell 集成
autosh init zsh

# 重新加载配置
source ~/.zshrc
